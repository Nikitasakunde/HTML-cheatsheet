This HTML CheatSheet is For You !
  1. HTML boilerplate: This is the basic html boilerplate code 
    
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>
        <!-- Body -->
</body>
</html>

    2. * Headings: There are six heading available in HTML, H1 is the largest among all, and H6 is smallest
  2. <h1> Tag 
    <h1>Heading 1</h1>
  3. <h2> Tag 
    <h2>Heading 2</h2>

    4. * Container:
  4. div tag 
    <div>This is div block</div>
  5. span tag 
    <span>This is span block</span>
  6. pre tag 
    <pre>Hello World</pre>
  7. code tag 
    <code>
import python
</code>

    8. * Text Formatting: Text formatting tags are used to format text or data of HTML documents. You cna do certain things like creating italic, bold ,strong to make your document look more attractive and understandable.
  8. <b> tag 
    <b>This is bold text</b>
  9. <strong> tag 
    <sub>I'am important text</sub>
  10. <sub> tag 
    <sub>Subscript</sub>

    11. * List : List can be either numerical, alphabetic, bullet, or other symbols.
  11. <ol> tag 
    <ol>
    <li>Data 1</li>
    <li>Data 2</li>
    <li>Data 3</li>
</ol>
  12. <ul> tag 
    <ul>
    <li>Data 1</li>
    <li>Data 2</li>
</ul>

    13. * Media: media is anything that is present in digital form such as image, video, audio,etc.
  13. <img> tag 
    <img src="Source_of_image" alt="Alternate text">
  14. <video> tag 
    <video width="480" height="320" controls>
<source src="demo_move.mp4" type="video/mp4">
